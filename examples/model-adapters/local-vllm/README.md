# Local vLLM adapter profile example

This is a declarative, interface-only example for Syzygy's constrained model-adapter API. It
describes an exact literal-loopback origin and OpenAI Responses route. It contains no executable,
credential, prompt, response, installer, or network client and does not start or contact vLLM.

Passing contract certification proves only schema, semantic profile, package containment, hostile
profile fixtures, and exact endpoint allow/deny probes. Runtime transport conformance and model
capabilities require separate fake-network and opt-in live evidence.

The endpoint shape reflects vLLM's public compatible-server documentation checked on 2026-07-15:
https://docs.vllm.ai/en/stable/serving/openai_compatible_server/
